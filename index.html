В 2021-м году порядок действий для входа в тб следующий: Воспользоваться моим скриптом автоустановки Flash Player и преднастроенного Firefox (только для Windows 7 x64 и новее).

Либо настроить то же самое, но самостоятельно:
  • Установите поддерживающую Flash Player версию браузера. Я рекомендую Firefox, т.к. ESR версия браузера продолжает получать обновления.
    Firefox < 85 (ESR 78 Win64/Win32/Mac) или Chromium < 88 (Win/Mac/Linux);
  • Установите крайнюю версию Flash Player без таймбомбы. Вам потребуется скачать деинсталлятор для удаления установленного флеша и, собственно, установщик для вашей связки ОС+браузер.
    Последняя версия Flash Player 32 без таймбомбы для win/mac/linux;
  • Исправьте авторизацию через официальный сайт следующим образом:
    • Установите расширение ModHeader для своего браузера (для Chrome/для Firefox);
    • Откройте окно импорта настроек, проследовав по следующим кнопкам:
    • Вставьте в текстовое поле эти настройки:
      [{"alwaysOn":true,"appendMode":"comma","backgroundColor":"#a1461d","filters":[{"comment":"","enabled":true,"resourceType":[],"type":"urls","urlRegex":".*games.itsrealgames.com/oauth.*"}],"headers":[],"hideComment":true,"respHeaders":[{"comment":"","enabled":true,"name":"Content-Type","value":"text/html"}],"shortTitle":"1","textColor":"#ffffff","title":"Profile 1","urlReplacements":[]}]
